## Erlang telegram bot

##### Instruction
To work you need a token. To get the token follow the instructions here [how-do-i-create-a-bot] (https://core.telegram.org/bots/#3-how-do-i-create-a-bot)

##### Run:
	make
	erl -pa ebin deps/jiffy/ebin/ -s raboter

#####Technology:
- Erlang
